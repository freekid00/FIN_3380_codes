Wrap up the above task to a user defined function, so that it can be used by foreach in later steps.

jtest = function(t1.name, t2.name, df) {
  df = df[which(df$permno %in% c(t1.name,t2.name)),]
  df = spread(df,permno,logprice)
  df = df[complete.cases(df),]
  ts = df[,c(2,3)]
  j = summary(ca.jo(ts))
  r = data.frame(stock1 = t1.name, stock2 = t2.name, 
                 stat = j@teststat[2])
  r[, c("pct10", "pct5", "pct1")] = j@cval[2, ]
  return(r)
}

Step 6. Create buy sell signals and calculate returns

Assume the spread is is Gaussian distributed. That is

\[Spread_t\sim N(0,\sigma^2)\]

* Calculate the standard deviation of the spread.
sd.spread = sd(goodpair$spread)

Select a cutoff of \(spread = \sigma^2\), if \(log(Y)-\alpha-\beta log(X)>\sigma\), buy \(\beta\) shares of X for and sell 1 share of Y, instead, buy 1 share of Y and sell \(\beta\) shares of X.

If buy 1 share of Y and sell \(\beta\) shares of X, the portfolio value at t is

\[ log(Y_t)-\beta \times log(X_t) = spread_t + \alpha \]

If buy 1 share of Y and sell \(\beta\) shares of X, the portfolio value at t+1 is

\[ log(Y_{t+1})-\beta \times log(X_{t+1}) = spread_{t+1} + \alpha \]

The return from t to t+1 is

\[ spread_{t+1}-spread_{t} \]

goodpair = goodpair %>% mutate(signal = ifelse(spread>sd.spread,-1,1))
barplot(goodpair$signal,col="blue",

Step 8. Model the spread as AR(1) process.

\[Spread_t = \theta Spread_{t-1} + \epsilon_t\]

With \(\epsilon_t\sim N(0,\sigma^2)\), the intercept is 0 since the \(Spread_t\) is constructed from regression.

* Recursively make 1-step forcast of the \(E(Spread_{t+1}|F_t)\): \(E(Spread_{t+1}|F_t) = \theta\times Spread_t\)
* Remember: The 1-step forecast error of AR(1) is \(\sigma\), that is \(Var(Spread_{t+1}-E(Spread_{t+1}|F_t)) = \sigma^2\)

Reconstruct the trading strategy: If the realised spread is much higher than the predicted spread, Y is too expensive/X is too cheap. Sell Y and buy X.

* If \(Spread_{t} - E(Spread_t|F_{t_1})>\sigma\), buy \(\beta\) shares of X for and sell 1 share of Y, instead, buy 1 share of Y and sell \(\beta\) shares of X.
* Estimate the AR(1) model
