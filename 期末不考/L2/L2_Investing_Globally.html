Step 4. Visualise the returns of all indices using package “ggplot2”

Use the ggplot2 package to plot the multiple lines

ggplot(df,aes(x,y,color)) + geom_line()/geom_point()/geom_bar() + labs(x,y,title) + theme +…

  • ggplot takes the a data.frame as input(df)
  • aes(x,y,color) set up the x and y values in plot, when ploting multiple lines, color is the classification column
  • geom_line()/geom_point()/geom_bar() indicate the plot type
  • labs(x,y,title) gives the x, y, title names
  • theme() to set the legend, background color, location of legend, etc…
library(ggplot2)
p = ggplot(ret.index,
           aes(x=as.Date(year,"%Y"),

Step 6. Calculate the cumulative returns

We use tapply function rather than the group_by function from dplyr

  • The tapply function is useful when we need to break up a vector into groups defined by some classifying factor, compute a function on the subsets, and return the results in a convenient form.
  • You can specify multiple factors as the grouping variable
  • When the results are vectors, the returned results from tapply is an array
ret.arr = tapply(ret.index$ret, INDEX=ret.index$tic, 
         FUN=function(x) cumprod(x+1))

Step 10. Build up a portfolio that maximise the Sharpe Ratio

Use fportfolio package, which search over different combinations of weights and find the weights that maximize the portfolio sharpe ratio

  • To use optimiser, we need to make sure that our covariance matrix of returns is positive-definite and has no NA values.
  • If this is not true, we won’t be able to run the optimization procedure.
  • Use the function is.positive.definite() from package matrixcalc to check the positive definite
library(matrixcalc)
# check if covariance matrix is positive definite
print(is.positive.definite(cov(ret.mat)))
## [1] TRUE
# check for na
print(any(is.na(ret.mat)))
## [1] FALSE
  • In fact, the covariance are always positive definite unless one variable is an exact linear function of the others, which is unlikely in real data

Use function tangencyPortfolio() to get the portfolio with highest sharpe ratio.
